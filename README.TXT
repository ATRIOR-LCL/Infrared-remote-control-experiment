实验器材:
	NANO STM32F4开发板V1版本
	
实验目的:
	学习1.3寸TFTLCD的使用,驱动TFTLCD屏.
	
硬件资源:
	1,DS0(连接在PC0) 
	2,1.3寸TFTLCD屏
   
    TFTLCD屏与NANO STM32F4开发板连接关系
	SCK  --------------------PA5 
	SDA ---------------------PA7
	WR --------------------- PA4
	CS-----------------------PA6
	PWR----------------------PA2
	RESET--------------------PA3
	VCC----------------------3.3V
	GND----------------------GND
	
实验现象:
	本实验通过LCD显示图像和字符。
